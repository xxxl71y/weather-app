# Weather Now

实时天气查询网页应用，浏览器打开即用。

## 功能

- **自动定位**：GPS 精确定位 + IP 定位后备
- **实时天气**：温度、天气状况、湿度、风速、降水量
- **动态背景**：晴天 / 多云 / 雨天 / 雪天 / 雷暴 自动切换主题
- **天气粒子**：雨滴、雪花 Canvas 动画
- **无需注册**：所有 API 免费免 Key

## 使用的免费 API

| 服务 | 用途 |
|------|------|
| [Open-Meteo](https://open-meteo.com/) | 全球天气预报 |
| [BigDataCloud](https://www.bigdatacloud.com/) | 坐标反查地址 |
| [ipapi](https://ipapi.co/) | IP 定位（后备） |

## 本地运行

浏览器直接打开 `index.html` 即可。
